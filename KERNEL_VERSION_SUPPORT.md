# 内核版本支持表

本文档记录当前自动构建矩阵中枚举的 `android_version`、`kernel_version`、`sub_level`、`os_patch_level` 组合。自定义构建也建议按这些组合填写。

说明：

- 自动构建入口：`.github/workflows/main.yml`
- 分版本矩阵：`.github/workflows/kernel-a*.yml`
- 自定义构建入口：`.github/workflows/kernel-custom.yml`
- 实际校验位置：`.github/workflows/build.yml`
- LTS 构建统一填写：`sub_level=X`，`os_patch_level=lts`。只有表中列出 `X/lts` 的内核线才在自动构建矩阵中启用。
- Android 12 / 5.10 需要填写 `revision`，其它内核线 `revision` 留空即可。
